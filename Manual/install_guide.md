# Superscan インストールガイド

## システム要件
- OS: Windows 10 / 11
- ドライブ: **Dドライブが必要** (OCRエンジン用。空き容量 5GB以上)
- ソフトウェア: Python 3.8以上 (パスが通っていること)

## 手順
1. **ファイルの展開**: ダウンロードしたファイルを任意のフォルダに展開します。
2. **インストールの実行**: `install.bat` を実行します。
   - 黒い画面（コマンドプロンプト）が開きます。
   - 自動的に `D:\AntigravitySupport\Superscan_venv` が作成されます。
   - 「Installation Complete!」と表示されるまで待機してください。
3. **拡張機能の登録**:
   - Chrome の設定メニュー -> 拡張機能 -> デベロッパーモード ON。
   - `extension` フォルダをドラッグ＆ドロップして登録します。

## 起動確認
- `start_app.bat` を実行し、画面右下の通知領域にアイコンが出れば成功です。
